# Webpage Cloning Requirements

1. **Visual Accuracy**: The cloned page must look exactly the same as the original website provided.
2. **Main Page Only**: No functional hyperlinks are needed; only the main landing page should be functional.
3. **Local Resources**: All images, scripts, and stylesheets must be stored locally. The page must function without an internet connection.
4. **Automation Script**: Keep a `.py` script (e.g., `download_site.py`) in the folder for future reference and refreshing.
5. **Clean HTML**: Remove unnecessary blank rows and whitespace from the final HTML.
6. **Localization**: Lock the page to the English language and remove other language options or switchers.
7. **Structure**: Create the clone as a subfolder within the repository, not as a separate repository.
